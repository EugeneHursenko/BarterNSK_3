Установка
=========

Установит все зависимости из requirements.txt

```shell
# windows
./install.cmd
```

Запуск на локальной машине для разработки

```shell
# windows
./run.cmd
```

Работа с консольными командами
==============================

Получения списка всех доступных команд
```shell
flask
```

Создание нового пользователя
```shell
flask user-create
```
